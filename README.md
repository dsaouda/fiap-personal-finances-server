## Instalação do MYSQL (Ubuntu)

```bash
apt-get install mysql-server
```
Obs: Para outros sistemas https://www.mysql.com/downloads/
 

## Import database do projeto

```bash
mysql -uroot -p -D gastos < db/database.sql
```
